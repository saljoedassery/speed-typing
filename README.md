### Try out

To try locally 

First you need some dependancies to install

- **NodeJs**
- **npm**

Then clone the repository

```sh
$ git clone https://github.com/saljoedassery/speed-typing.git
$ cd speed-typing
$ npm install
$ npm start
```

it will open in localhost:3000
